# Энтропийный оптимальный транспорт: анализ двойственности и методы оптимизации

Данный проект выполнен в рамках математического практикума ФПМИ МФТИ. Работа посвящена глубокому теоретическому исследованию задачи энтропийного оптимального транспорта (EOT) и применению современных методов оптимизации для её решения.

## 📄 Теоретическая часть (Отчет)

Вся основная суть работы, математические выкладки и доказательства содержатся в файле **[`optimal-transort-optimization.pdf`](optimal-transort-optimization.pdf)**. 

**Ключевые темы, разобранные в отчете:**
* Постановка классической транспортной задачи и переход к энтропийно-линейному программированию (ЭЛП). 
* Построение двойственной задачи и анализ её свойств (выпуклость, липшицевость градиента, отсутствие сильной выпуклости).
* Анализ классического градиентного спуска: вывод леммы о достаточном убывании и оценка скорости сходимости.
* **Primal-Dual подход** (на основе работ П. Двуреченского, А. Гасникова и А. Крошнина): решение проблемы осцилляций метода Нестерова через усреднение транспортного плана. Доказательство оценки сложности $\mathcal{O}(1/\varepsilon)$.
* **Методы второго порядка:** анализ применимости метода Ньютона, обход проблемы вырожденности матрицы Гессе с помощью кубической мажоранты (Кубический Ньютон).
* **Соовременные методы второго порядка. Анализ статей 2024 года:**  Исследование структуры Гессиана, использование дополнения Шура и метода сопряженных градиентов (CG) для снижения вычислительной сложности. Анализ самосогласованности и демпфированного шага Ньютона.

## 💻 Практическая реализация

В файле [**experiments.ipynb**](experiments.ipynb) представлена программная реализация некоторых описанных в отчете методов. На примере синтетической 1D-задачи (перемещение массы между дискретизованными Гауссовскими распределениями) реализованы следующие алгоритмы:

1. Классический градиентный спуск (GD).
2. Ускоренный метод Нестерова (NAG).
3. Primal-Dual метод с усреднением транспортного плана.
4. Метод кубического Ньютона.

В ноутбуке также приведено сравнение эмпирической скорости сходимости методов и визуализация итогового транспортного плана.

## 🛠 Установка и запуск

Для запуска практической части на локальной машине выполните следующие шаги в терминале:

1. Склонируйте репозиторий:
```bash
git clone https://github.com/ВАШ_НИК/optimal-transport-optimization.git
cd optimal-transport-optimization
```

2. Установите необходимые зависимости (рекомендуется использовать виртуальное окружение):
```bash
pip install numpy scipy matplotlib jupyter
```

3. Запустите Jupyter Notebook:
```bash
jupyter notebook experiments.ipynb
```
